# ejercicio:3.0.0

Aplicación básica de suma desarrollada en Python.

## Requisitos

- Python 3.14
- Docker

## Instalar dependencias

```bash
pip install -r requirements.txt
```

## Ejecutar aplicación

```bash
python app.py
```

## Ejecutar pruebas

```bash
pytest
```

## Construir Docker

```bash
docker build -t ejercicio:3.0.0 .
```

## Ejecutar Docker

```bash
docker run ejercicio:3.0.0
```
